# 说明

* Demo中有三种效果，演示图中第一个为BookPageTransformer，第二个为ScalePageTransformer，第三个为CubePageTransformer
* 三种效果文件可直接拷贝，没有资源文件，遂没有封装为库
* 使用方式如：mViewPage.setPageTransformer(false, new BookPageTransformer());
